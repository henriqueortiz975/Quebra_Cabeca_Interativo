Frontend Inteligente (Evolução com JavaScript)

Aprender a manipular classes e elementos no DOM com JavaScript, usando eventos para controlar estados dinâmicos

Manipulação de Classes

Permite alterar classes CSS via JavaScript.

// Adiciona a classe "ativa" ao elemento
elemento.classList.add("ativa");

// Remove a classe "ativa"
elemento.classList.remove("ativa");

// Alterna a classe:
// se existir → remove
// se não existir → adiciona
elemento.classList.toggle("ativa");

Estado Dinâmico

O estado passa a ser controlado pela classe

// Antes usamos:
let ativo = false;

// Agora usamos o próprio DOM:
// Verifica se o elemento possui a classe
elemento.classList.contains("ativa");

Múltiplos Elementos

📌 querySelectorAll

// Seleciona todos os elementos com classe "peca"
const pecas = document.querySelectorAll(".peca");

📌 forEach

// Percorre todos os elementos encontrados
pecas.forEach(function(peca) {

  // Código executado para cada peça

});

Eventos em Vários Elementos

// Para cada peça
pecas.forEach(function(peca) {

  // Adiciona evento de clique
  peca.addEventListener("click", function() {

    // ação ao clicar

  });

});
