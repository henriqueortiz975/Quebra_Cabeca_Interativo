Exemplicando como e sobre o JavaScript e DOM

Variáveis

Espaços na memória para guardar dados, que também é usada para armazenar informações temporárias.

Tipos de Variáveis

Tipo Escopo Pode alterar?
var Função Sim
let Bloco Sim
const Bloco Não

JavaScript

  • Linguagem de programação para web
  • Executado no navegador
  • Adiciona interatividade e dinamismo
  • Altera HTML e CSS via código

Tipos de Dados no JavaScript

você não precisa declarar o tipo da variável. Ele é dinamicamente atribuído de acordo com o valor que você coloca nela. Isso faz parte de uma característica da linguagem chamada tipagem dinâmica.

Tipo Exemplo
String "Olá mundo"
Number 10 / 3.14
Boolean true / false
Array [1, 2, 3]
Object {nome: "Ana", idade: 22}
null null
undefined variável não inicializada
function function() {}

DOM

  • DOM significa Document Object Model (Modelo de Objeto do Documento)
  • É a forma como o navegador transforma o HTML em uma estrutura manipulável pelo JavaScript.
